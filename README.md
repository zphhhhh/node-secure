# 演示 Web 安全（XSS 与 CSRF 攻击）

### 如何运行

1. 安装 `Node.js`
2. 进入项目目录并运行 `npm install`
3. 运行 `npm start`，在浏览器访问 `http://localhost:3000` 即可

### 操作方法

1. 演示 XSS 攻击：`http://localhost:3000`
2. 演示 CSRF 攻击：`http://localhost:3000/weibo` 及 `http://localhost:3000/ituring`
